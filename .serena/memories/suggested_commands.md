# Suggested Commands
- `git status --short` to inspect working tree changes.
- `rg -n "pattern" .` for fast repo-wide text search on Windows PowerShell.
- `Get-Content -Path <file> -TotalCount <n>` to read targeted file sections.
- `./tests/claude-code/run-skill-tests.sh` to run the fast Claude Code skill test suite.
- `./tests/claude-code/run-skill-tests.sh --integration` for slow integration coverage.
- `./tests/claude-code/run-skill-tests.sh --test <test-file>.sh` to run a single skill test.
- `./tests/claude-code/run-skill-tests.sh --verbose` for more detailed test logs.
